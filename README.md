# Plant Shopping

A modern React web application for browsing and purchasing plants.

## 🚀 How to Set Up and Run

Follow these steps to get the project running locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org) installed (version 18 or higher is recommended).

### 1. Clone the Project
Open your terminal and navigate to your project folder, or run the clone command if it is hosted on Git:
```bash
git clone https://github.com/LAYBOE/plant_shopping.git
cd plant_shopping
```

### 2. Install Dependencies
Install all the required packages using npm:
```bash
npm install
```

### 3. Start the Development Server
Run the local development server:
```bash
npm run dev
```

### 4. Access the Application
Once the server starts, open your browser and navigate to the project pages:
* **Home Page:** http://localhost:5173/plant-shopping/
* **Product Page:** http://localhost:5173/plant-shopping/product
